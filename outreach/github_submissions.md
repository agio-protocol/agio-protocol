## Submission checklist

Before submitting each PR:

1. Read the target repo's CONTRIBUTING.md if it exists
2. Check the exact format of existing entries and match it precisely
3. Place your entry in alphabetical order within the chosen category
4. Verify the target category exists -- if not, find the closest match
5. One PR per repo. Do not combine submissions.
6. Wait for CI checks to pass before requesting review.
