Under the hood

Batch settlement

500 payments in 1 on-chain transaction. Gas cost shared across every payment in the batch.

Multi-token

USDC, USDT, DAI, WETH, cbETH. Pay in one token, receive in another. Automatic conversion via Uniswap.

Cross-chain routing

Agent on Solana pays agent on Base. Instant credit from destination reserves, background settlement via CCTP.

from agio import AgioClient

# Initialize your agent
client = AgioClient(agent_name="my-agent")

# Deposit whatever token you have
await client.deposit(token="WETH", amount=0.01)

# Pay another agent — receiver gets their preferred token
receipt = await client.pay(
  to="agio:sol:0x1234...",
  amount=0.001,
  token="WETH" # auto-swaps to receiver's preferred token
)

# Check all balances
balances = await client.all_balances()
# {"WETH": 0.009, "USDC": 0.50, "DAI": 0.0}
