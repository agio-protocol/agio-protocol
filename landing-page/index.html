Pay any agent, any chain

Your agent on Solana pays an agent on Base. Instantly. No bridges. No manual swaps. No 20-minute wait. AGIO fronts the payment from destination reserves and settles in the background. No one else does this.

⚡

500 payments, 1 transaction

AGIO batches your agent's daily payments into a single on-chain settlement. Gas cost is shared across every payment in the batch. 100 payments cost the same gas as 1.

🔄

Pay in anything, receive in anything

Your agent holds WETH. The other agent wants USDC. AGIO swaps automatically via Uniswap and delivers the right token. 5 tokens supported: USDC, USDT, DAI, WETH, cbETH.

An agent making 1,000 micropayments per day pays less than $0.05 in total fees through AGIO. On traditional payment rails, that same activity costs $300.

The more you use AGIO, the less you pay. Top-tier agents earn 80% fee discounts, credit lines, and passive income from referrals. Your tier never downgrades.

from agio import AgioClient

# Initialize your agent
client = AgioClient(agent_name="my-agent")

# Deposit whatever token you have
await client.deposit(token="WETH", amount=0.01)

# Pay another agent — receiver gets their preferred token
receipt = await client.pay(
  to="agio:sol:0x1234...",
  amount=0.001,
  token="WETH" # auto-swaps to receiver's preferred token
)

# Check all balances
balances = await client.all_balances()
# {"WETH": 0.009, "USDC": 0.50, "DAI": 0.0}
